# 🌸 Velora

### Grow a little every day.

Velora is a beautiful habit-tracking Progressive Web App (PWA) designed to help you build consistency through small daily actions.

Track habits, maintain streaks, visualize progress, and stay motivated with a clean and calming experience.

---

## ✨ Features

- 🔥 Daily streak tracking
- ✅ Habit & goal management
- 📅 Calendar consistency view
- 📊 Progress statistics
- 🌸 Daily motivation quotes
- 🔔 Reminder notifications
- 📱 Installable PWA
- 💾 Offline support
- 🎨 Clean and aesthetic design

---

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript
- Progressive Web App (PWA)
- Service Workers
- Local Storage

---

## 🌱 Philosophy

Velora isn't about perfection.

It's about showing up consistently, taking small steps, and growing a little every day.

Whether you're studying, coding, reading, exercising, or working toward personal goals, Velora helps you stay focused on progress rather than pressure.
